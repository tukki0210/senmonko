ITシステム科

プログラミング、データベース、Webシステム開発などの実習と、ITパスポート・基本情報技術者・Oracle認定Javaプログラマー試験等の対策授業を行います。これらにより、将来、ITエンジニアとして活躍できる人材を育成します

また、IT技術はすべての業界で必要とされる技術です。IT技術を学び直し、これまで従事した実務経験のある分野でデジタル化の担い手として再就職を目指す方もいます。

主な授業・カリキュラム

Javaプログラミング

Javaプログラミング言語の基本を学びます。オブジェクト指向プログラミングの概念、変数、制御構造、クラス・メソッドの使い方など、プログラミング初心者にもわかりやすく指導します。実際にコードを書きながら、基礎からしっかり学習し、アプリケーション開発の第一歩を踏み出します。

関係するテーマ
  • Java言語
  • オブジェクト指向
  • Oracle認定Javaプログラマー
    Bronze・Silver試験対策

Web制作

この授業では、Webサイト制作の基本を学びます。HTML、CSS、JavaScriptを使用し、レスポンシブデザインやユーザーインターフェースの基礎を習得します。実際にWebページを作成しながら、フロントエンド技術を体系的に学び、現場で役立つスキルを実践的に身につけることができます。

関係するテーマ
  • HTML・CSS
  • JavaScript
  • WordPress
  • Webクリエイター能力認定試験対策

Webアプリケーション開発

PHPというプログラミング言語と、Laravelというフレームワークを使用して、Webアプリケーションの開発を学びます。PHPの基本的な文法やデータベースとの連携方法を習得し、Laravelを使って効率的にWebシステムを構築するスキルを養います。現場でよく使用される開発フレームワークを実践的に学び、スピーディーかつ保守性の高いWeb開発を実現します。

関係するテーマ
  • PHP
  • Laravel
  • Web開発
  • ポートフォリオ作成

サーバ構築

データベースの基本概念や仕組みを学びます。リレーショナルデータベース、SQL（データベース操作言語）の基礎、データの保存・検索方法、データベース設計の基礎を実践的に学習します。簡単なデータベースを構築・操作し、効率的なデータ管理の方法を習得することで、業務に直結するスキルを身につけます。

関係するテーマ
  • Linuxコマンド
  • サーバ構築・設定
  • 仮想化
  • 開発環境
  • クラウド技術・AWS

基本情報技術者試験対策

基本情報技術者試験に合格するための知識とスキルを学びます。試験範囲に含まれるIT基礎理論、アルゴリズム、ネットワーク、データベース、セキュリティなど、幅広い分野を網羅的にカバーし、過去問題や模擬試験を通じて実践的な力を養います。また、効率的な学習方法や解答テクニックも指導し、試験に向けた万全の準備ができるようサポートします。国家資格の取得により、IT業界でのキャリアアップを目指します。

主なテーマ
  • 情報理論基礎
  • アルゴリズム
  • データベース
  • ネットワーク
  • セキュリティ
  • システム開発入門
  • 簿記入門など
